API Examples

On these pages are several examples of accessing various APIs and retrieving data. You will note that they are all very similar to each other.

Some of these calls use XMLHttpRequest objects to call the API. Where this is not supported by the API, a JSONP example is used. Simple examples of each type of call are shown below:

XMLHttpRequest with EventListener

One way of using an XMLHttpRequest is to add an EventListener which will fire when the XMLHttpRequest 'loads'. An example of doing this is here.

XMLHttpRequest and onreadystatechange

Another way of using an XMLHttpRequest is to set the onreadystatechange property to be a function which can process the API response. An example of doing this is here.

JSONP Example

XMLHttpRequest objects allow us to access APIs that have Cross Origin Resource Sharing (CORS) enabled. Not all APIs allow this. Some of those that do not allow CORS do allow JSONP. An example of using JSONP can be found on this page
